# Task: Hello Greeting

Write a function `greet` that takes a name (String) and returns a greeting string.

## Signature

```
fn greet(name: String) -> String
```

## Behavior

- `greet("World")` returns `"Hello, World!"`
- `greet("LM")` returns `"Hello, LM!"`
- `greet("")` returns `"Hello, !"`

## Notes

- Use `++` for string concatenation (not `+`).
- The function should be pure (no IO).
