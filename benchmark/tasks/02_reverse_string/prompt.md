# Task: Reverse String

Write a function `reverse` that reverses a string character by character.

## Signature

```
fn reverse(s: String) -> String
```

## Behavior

- `reverse("hello")` returns `"olleh"`
- `reverse("ab")` returns `"ba"`
- `reverse("")` returns `""`
- `reverse("a")` returns `"a"`

## Notes

- LM has no string indexing or character operations built in.
- This task requires string indexing features not yet available in the current interpreter.
