# Task: Reverse String

Write a function `reverse` that reverses a string character by character.

## Signature

```
fn reverse(s: String) -> String
```

## Behavior

- `reverse("hello")` returns `"olleh"`
- `reverse("ab")` returns `"ba"`
- `reverse("")` returns `""`
- `reverse("a")` returns `"a"`

## Notes

- Use `str_len(s)` for the character count.
- Use `char_at(s, i)` to read the character at index `i`.
