# Task: Fibonacci

Write a function `fib` that returns the Nth Fibonacci number using recursion.

## Signature

```
fn fib(n: Int) -> Int
```

## Behavior

- `fib(0)` returns `0`
- `fib(1)` returns `1`
- `fib(2)` returns `1`
- `fib(5)` returns `5`
- `fib(10)` returns `55`

## Notes

- Use the standard definition: fib(0)=0, fib(1)=1, fib(n) = fib(n-1) + fib(n-2).
- LM has no loops; use recursion.
- The function should be pure.
