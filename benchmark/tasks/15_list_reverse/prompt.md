# Task: List Reverse

Write a function that reverses a list.

## Signature

```
fn reverse(lst: List<Int>) -> List<Int>
```

## Behavior

- Reverse `[1,2,3]` returns `[3,2,1]`
- Reverse `[]` returns `[]`
- Reverse `[1]` returns `[1]`

## Notes

- Use recursion to build the reversed list.
- This task requires list builtins in the type checker.
